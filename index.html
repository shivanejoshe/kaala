The Kaala Stack
How it works
Each layer is independently testable and publicly documented.
Layer 01

Swiss Ephemeris

Sub-arc-minute planetary positions via pyswisseph. Lahiri ayanamsa for sidereal coordinates. The same computational core used by professional Vedic software.

Layer 02

Vimshottari Dasha Engine

Full 120-year dasha timeline computed from natal Moon nakshatra. Mahadasha → antardasha → pratyantara (3 levels). Used to determine the operating period of any chart.

Layer 03

Transit Overlay

Current planetary positions overlaid on natal chart. Computes Vedic special aspects, Sade Sati phases, Guru transit house position, and nodal activations.

Layer 04

Calibrated Signature Scoring

9 astrological signatures scored 0–1.0 each. Weights derived from Monte Carlo calibration against 300+ historical events. Generic weights replaced by chart-specific empirical lifts. Research Note #005.

Layer 05

Confluence Engine

Weighted sum of active signatures → a single confluence score (0–100%). Windows above threshold where ≥1 validated signature fires are surfaced as predictions.

Layer 06

Public Scoring & Falsifiability

Every prediction has a specific date, a specific claim, and a public tracker. The system is wrong until proven otherwise. Hits and misses are both recorded.
